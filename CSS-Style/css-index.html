What is DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing programming languages to manipulate the content, structure, and style of the document.

HTML class and Id attributes

When an HTML element is given an id, it serves as a unique identifier for that element.
On the other hand, when a HTML element is given a class, it now belongs to that class. More than one element can belong to a single class, but every element must have a unique id(if assigned).

we can add multiple classes to an element like this:
<div id="uniqueId" class="c1 c2 c3"> </div>
